# Live-Code-Editor

**Developer:** Kavisha Mishra

A **browser-based Live Code Editor** that allows users to write **HTML, CSS, and JavaScript** code and see real-time output instantly.
This project demonstrates **frontened development skills**, including responsive design, modern UI styling, and live preview using vanilla JavaScript, HTML, and CSS.

_ _ _

## Features

1. Live HTML, CSS, and JavaScript preview
2. Fully responsive and modern UI
3. Clean and easy-to-use interface for learning and experimentation
4. Screenshot included to showcase functionality

_ _ _

## Tech Stack

1. **HTML** - structure of the editor
2. **CSS** - styling and responsive layout
3. **JavaScript** - real-time code rendering

## How to use

1. Open 'index.html' in a modern browser or use **Live Server** in VS Code
2. Type HTML, CSS, Or JS in the editor sections
3. See changes appear instantly in the preview area
